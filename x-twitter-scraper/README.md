# x-twitter-scraper

Xquik skill for X/Twitter API automation, tweet search, profile tweets, follower export, media download, posting, replies, webhooks, MCP, and SDK workflows.

## What it does

Use this skill when Claude Code needs to:

- Search tweets with keywords, hashtags, accounts, or advanced Twitter search operators
- Get profile tweets, user timelines, liked tweets, media tweets, and engagement metrics
- Export followers, following, retweeters, likers, replies, quotes, and mentions
- Download tweet media and collect hosted media URLs
- Monitor accounts and deliver HMAC-signed webhook events
- Send tweets, post replies, like, repost, follow, DM, or update profiles after explicit approval
- Build integrations with REST API, MCP tools, and official SDKs

## Installation

Part of [rbartoli/agent-skills](https://github.com/rbartoli/agent-skills). Install through the marketplace:

```bash
/plugin marketplace add https://github.com/rbartoli/agent-skills
/plugin install x-twitter-scraper@rbartoli-agent-skills
```

Or install from the upstream Xquik skill repo:

```bash
npx skills add Xquik-dev/x-twitter-scraper --skill x-twitter-scraper
```
